A front-end-focused role on a product that helps users manage healthcare-related payments through a wallet-like experience. The codebase was primarily React and Next.js, with a Java back end.

Most of my contributions were on the front end. I occasionally picked up back-end tickets — small fixes and unit tests written with Mockito — but the bulk of the work was in the UI.

Because the user base included older adults, accessibility was a real part of how we built features. We were intentional about semantic HTML, screen reader behaviour, and keeping flows simple and understandable.
